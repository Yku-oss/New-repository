### MyBatis
## 概念
````
MyBatis是一个Java持久层的一个框架，是一个能让Java代码里的sql语句连接数据库操作，能极大提高开发效率
````

## 作用
````
可以将sql语句和java代码进行“解耦”，能直接连接数据库，可以专注的写sql本身，Mybatis会自动将数据打包成一个对象放入到数据库中
再将结果映射成对象返回给java代码输出。
````

## 框架理解
````
Mybatis 本身就是一个半自动的ORM(对象关系)框架

sql语句必须手写，但Mybatis会负责将sql的输入参数自动映射成java对象，并把查询结果自动映射成回java对象

半自动的好处就是后面可以进行对sql语句的调优。
````

# 工作流程
````
      定义一个Mapper（数据访问层）接口，里面定义了数据操作的方法
                        👇
      写好一个xml文件的sql语句，并于Mapper接口连接
                        👇
      Mybatis会自动将调用的操作方法与sql语句进行配对，
      然后翻译xml文件中的sql语句，并自动将结果映射成java对象返回
````
