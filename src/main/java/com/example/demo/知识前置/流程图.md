## 流程图
```
        浏览器发起 HTTP 请求或者是javafx等前端页面
            ↓
        【1】Tomcat（内置 Web 服务器）接收请求
            ↓
        【2】Spring MVC 的 DispatcherServlet（前端控制器）接管
            ↓
        【3】根据 @GetMapping 找到对应的 Controller 方法
            ↓
        【4】Controller 调用 Service 层方法（业务逻辑）
            ↓
        【5】Service 调用 Mapper 层方法（数据访问）
            ↓
        【6】MyBatis 执行 SQL，查询/操作数据库
            ↓
        【7】数据库返回结果
            ↓
        【8】MyBatis 将 ResultSet 映射成 Java 对象
            ↓
        【9】Mapper 返回对象给 Service
            ↓
        【10】Service 返回处理结果给 Controller
            ↓
        【11】Controller 将对象转为 JSON（通过 HttpMessageConverter）
            ↓
        【12】DispatcherServlet 将 JSON 写回 HTTP 响应体
            ↓
        浏览器收到 JSON 响应


```
